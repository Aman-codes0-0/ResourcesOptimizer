COMMON ISSUES AND SOLUTIONS:

ISSUE: Server won't start
SOLUTION:
   1. Check if port 3000 is already in use
   2. Run: netstat -ano | findstr :3000
   3. Kill the process or change port in server.js
   4. Verify Node.js is installed: node --version

ISSUE: "Cannot find module" error
SOLUTION:
   1. Delete node_modules folder
   2. Delete package-lock.json
   3. Run: npm install
   4. Restart server

ISSUE: Database not saving changes
SOLUTION:
   1. Check file permissions on resources.db
   2. Ensure disk space is available
   3. Restart the server
   4. Check for error messages in console

ISSUE: WebSocket not connecting
SOLUTION:
   1. Check browser console for errors
   2. Verify server is running
   3. Clear browser cache
   4. Try a different browser
   5. Check firewall settings

ISSUE: "No suitable resources found" when auto-allocating
SOLUTION:
   1. Check if resources are available (not at 100% utilization)
   2. Verify skill match (at least 50% required)
   3. Add resources with required skills
   4. Consider lowering skill requirements

ISSUE: Page not loading
SOLUTION:
   1. Clear browser cache (Ctrl+Shift+Delete)
   2. Hard refresh (Ctrl+F5)
   3. Check browser console for errors
   4. Verify server is running
   5. Try http://localhost:3000 directly

ISSUE: Real-time updates not working
SOLUTION:
   1. Check WebSocket connection status (bottom left)
   2. Refresh the page
   3. Restart the server
   4. Check browser WebSocket support
